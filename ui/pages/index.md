# PBPE Marketplace – Dashboard

The dashboard provides a high-level overview of the PBPE Marketplace.

## Key Sections

- Marketplace summary  
- Recent PBPE transactions  
- Featured projects  
- Key metrics (PBPE issued, settled, active projects)  

## Example Metrics (Conceptual)

- Total PBPE issued  
- Total PBPE settled  
- Active projects  
- Regions covered
