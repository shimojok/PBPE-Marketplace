# 3. PBPE‑UID (7‑Digit Universal Identifier)

PBPE‑UID is the **global address system for planetary value**.


Defines asset type:

- CRD – PBPE Credits

- INS – PBPE Insurance
    
- FRM – Farm / Producer
    
- ORG – Enterprise
    
- HLT – HealthBook
    
- MBT – MBT55 Biosecurity
    
- COF – Coffee Sector
    

### Sequence (1234)

Base36 (0–9, A–Z) → 1,679,616 unique combinations

- Scope 3 reporting
